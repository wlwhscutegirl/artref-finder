# Plan: dual-mode-sketch-search (듀얼 모드 + 스케치 검색)

## Phase 7 — ArtRef Finder

### 개요

앱을 크게 2가지 모드로 구분:
- **모드 A "마네킹 모드"**: 기존 3D 마네킹 기반 정밀 검색 (중급+ 아티스트)
- **모드 B "드로잉 모드"**: 직접 그린 스케치로 유사 실사 사진 검색 (초보자)

랜딩 페이지에서 모드 선택 → 각 모드별 전용 검색 페이지로 분기.

### 현재 상태 (As-Is)

- `/search` 단일 페이지에 마네킹 기반 검색만 존재
- 랜딩 페이지 Hero에서 "레퍼런스 검색하기" → `/search`로 이동
- 드로잉/스케치 입력 기능 없음

### 목표 상태 (To-Be)

- `/search` → `/mannequin` (마네킹 모드, 기존 기능 이동)
- `/sketch` (드로잉 모드, 신규)
- 랜딩 페이지에서 2가지 모드 카드로 선택
- 상단 헤더에서 모드 전환 탭
- 드로잉 캔버스 (펜/지우개/크기/색상/되돌리기)
- 이미지 업로드 (스케치 사진)
- AI 스케치 → 포즈/실루엣 추출 → 유사 실사 사진 매칭

## 기능 요구사항 (FR)

### FR-01: 듀얼 모드 네비게이션

- 랜딩 페이지 Hero 섹션: 2개 모드 선택 카드
  - 마네킹 모드: "3D 마네킹으로 정밀 검색" → `/mannequin`
  - 드로잉 모드: "그림을 그려서 쉽게 검색" → `/sketch`
- 상단 헤더: 모드 전환 탭 (마네킹 | 드로잉)
- 두 모드에서 공통 컬렉션/대시보드 접근 가능

### FR-02: 라우팅 변경

- `/search` → `/mannequin` (기존 검색 페이지 경로 변경)
- `/search` → `/mannequin`로 리다이렉트 (하위 호환)
- `/sketch` 신규 페이지

### FR-03: 드로잉 캔버스

- HTML5 Canvas 기반 자유 드로잉
- 도구: 펜 / 지우개
- 설정: 선 굵기 (1~20px), 색상 (검정 기본)
- 되돌리기(Undo) / 다시하기(Redo) (Ctrl+Z / Ctrl+Y)
- 캔버스 전체 지우기
- 캔버스 크기: 512×512 고정 (정사각형)
- 터치 드로잉 지원 (모바일/태블릿)

### FR-04: 스케치 이미지 업로드

- 파일 드래그앤드롭 또는 클릭 업로드
- 지원 포맷: PNG, JPEG, WebP
- 최대 5MB
- 업로드 이미지를 캔버스에 표시 (위에 드로잉 가능)

### FR-05: 스케치 → 포즈 추출 + 검색

- 캔버스/업로드 이미지에서 실루엣/엣지 추출
- 추출된 특징을 기존 포즈 벡터와 비교
- Phase 4의 AI 포즈 추출 파이프라인 재활용
- 추출 실패 시 (사람 형체 없음) → 태그 기반 fallback

### FR-06: 드로잉 모드 결과 표시

- 스케치와 유사한 실사 사진 그리드 (유사도 내림차순)
- 유사도 점수 배지 표시
- 결과 이미지를 컬렉션에 저장 가능
- "이 포즈로 마네킹 모드에서 열기" 버튼 (모드 전환)

### FR-07: 모드별 온보딩

- 마네킹 모드: 기존 온보딩 유지
- 드로잉 모드: "캔버스에 대충 그려보세요!" 간단 안내
- 모드 선택 시 처음이면 해당 모드 온보딩 표시
