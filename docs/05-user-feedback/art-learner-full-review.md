## 6. 핵심 인사이트 + 다음 단계 제안

### 핵심 인사이트

#### 1. C1/C2 수정의 임팩트가 명확하게 검증됨
- 해부학 오버레이 평균 점수: **3.95 → 4.33** (+0.38)
- C1(다중 선택) 만족도 2.5 → 4.7, C2(모바일 범례) 만족도 2.0 → 4.5
- 20명 중 19명이 "Critical 이슈가 해결되었다"고 확인
- 특히 교육자 그룹(한지민, 조민정)과 학습자 그룹(장서현, 신하영)에서 다중 선택의 학습 효과를 강하게 체감

#### 2. 2D 모드가 성능 문제를 해결했지만 기능 격차가 새로운 Critical 이슈로 부상
- 2D 폴백 뷰어 도입으로 저사양/모바일 사용자 이탈 문제는 완화됨
- 그러나 2D 모드에서 해부학/조명이 빠지면서 **"핵심 기능의 절반을 사용할 수 없다"**는 새 불만 발생
- 17/20명이 2D 해부학 미지원을 Critical로 분류

#### 3. 전체 기능 체험 후 NPS가 +25 → +40으로 대폭 상승
- 이전 리뷰는 해부학 중심이었으나, 전체 기능(포즈 매칭, 조명, AI 추출 등)을 경험한 후 추천 의향이 크게 증가
- 특히 **포즈 유사도 매칭 + 조명 시뮬레이션 + 태그 자동 추천** 3종 세트가 "다른 서비스에 없는 독보적 가치"로 인식

#### 4. 초급과 중급의 핵심 요구가 명확히 분리됨
- **초급**: 학습 도구 (해부학, 온보딩, 저사양 지원, 가격 접근성)
- **중급**: 작업 도구 (외부 툴 연동, 포즈 관리, 유사도 정밀도, 이미지 DB 다양성)

#### 5. 플랜 체계 개선이 전환율에 직결
- Free 100회가 학생/입시생에게 여전히 부족
- Student 2,900원이 학생 접근성에 적절한 가격대로 평가됨 (9/10 초급 긍정)
- Lite 4,900원은 동인 작가/인디 개발자 대상으로 전환 가능성 높음
