## PART 2: 그룹 토론 (테마별)

### 토론 1: "핵심 강점은 무엇인가?"

**김도윤(게임):** 솔직히 말해서 이 서비스의 킬러 피처는 **3D 포즈 → 실사 레퍼런스 매칭**이에요. 기존에는 머릿속 포즈를 텍스트로 검색했는데, 여기선 직접 관절을 꺾어서 "이 포즈랑 비슷한 사진 줘"라고 하면 바로 나와요.

**이준호(3D):** 동의합니다. 저는 ZBrush에서 모델링하다가 레퍼런스가 필요할 때 항상 Pinterest에서 30분씩 헤맸거든요. 여기선 마네킹 포즈 잡고 2분이면 끝나요.

**한소율(초상화):** 저한테는 **조명 시뮬레이션**이 핵심이에요. "측광 + 하드라이트"를 3D로 세팅하면 딱 그 느낌의 사진이 나와요. 렘브란트 라이팅 레퍼런스를 이렇게 쉽게 찾은 적이 없어요.

**임채원(강사):** 교육자 입장에서는 **교육 모드(안전 필터)**가 가장 큰 강점이에요. 미술 수업에서 인체 레퍼런스를 보여줄 때 항상 "이거 학부모가 보면 어쩌지" 걱정했는데, 교육 모드 켜면 완전 안전한 이미지만 나와서 수업에 바로 활용할 수 있어요.

**장동현(스토리보드):** 저는 **무드보드 내보내기**가 최고예요. 컬렉션 만들고 드래그로 배치하고 PDF 한 장이면 감독 브리핑 자료가 되거든요.

---

### 토론 2: "가장 불편한 점은?"

**배준혁(어시, 저사양):** 저는 솔직히 **저사양 기기에서 못 써요**. Three.js 렌더링이 제 노트북에서 5fps 나오고, 관절 하나 돌리는 데 1초씩 걸려요. 2D 심플 모드 같은 게 있으면 좋겠어요.

**김나연(스마트폰):** 맞아요, 저도 **모바일에서 3D 뷰어가 거의 사용 불가**예요. 스케치 모드는 괜찮은데, 메인 기능인 마네킹 모드를 스마트폰에서 못 쓰면 반쪽짜리죠.

**박서연(웹툰, 태블릿):** 태블릿에서는 돌아가긴 하는데 **터치로 관절 잡기가 너무 어려워요**. 손가락이 관절보다 큰데... 터치 영역이 더 넓었으면 좋겠어요.

**정민수(학생):** **무료 50회 검색**이 하루에 금방 차요. 과제할 때 태그 바꿔가며 검색하면 20분이면 다 써요. 학생 인증 할인이라도 있으면...

**강현우(인디):** 동의. 인디 개발자도 마찬가지. **Free → Pro 사이에 중간 요금제**가 있으면 좋겠어요. 월 5천원 정도면 바로 결제할 텐데.

**류하은(순수미술):** 저는 **이미지 다양성**이 아쉬워요. 인물 포즈 위주라서, 정물·풍경·추상적인 참고 자료는 거의 없어요.

**조성민(건축):** 맞아요, **환경/배경 레퍼런스**가 부족해요. "카페에 앉아있는 사람" 같은 건 찾을 수 있는데, 카페 인테리어 자체의 레퍼런스는 거의 없어요.

### 토론 3: "이 기능이 추가되면 돈 내고 쓰겠다"

**오태현(애니 감독):** 1순위는 **실시간 팀 협업**이에요. 지금 컬렉션 공유가 URL 기반이라 일방향인데, 여러 명이 동시에 무드보드를 편집할 수 있으면 팀 플랜 바로 결제합니다.

**최유진(패션):** **의상/소재 전용 검색 모드**가 있으면 좋겠어요. 포즈보다는 "빨간 실크 드레스의 주름이 어떻게 잡히는지"를 찾고 싶을 때가 많거든요. 3D 마네킹에 옷을 입힐 수 있으면 미쳐요.

**신예린(사진):** **EXIF 기반 상세 조명 정보**가 있으면 좋겠어요. "이 사진은 f/2.8, ISO 400, 85mm 렌즈로 찍었다" 같은 촬영 정보를 레퍼런스 옆에 같이 보여주면, 사진 재현할 때 엄청 유용할 거예요.

**송지원(VTuber):** **캐릭터 시트 자동 생성**이요. 포즈 여러 개 잡아놓으면 정면/측면/후면/3/4 뷰를 자동으로 그리드로 만들어주는 기능. 커미션 작업할 때 필수거든요.

**이재훈(피규어):** **해부학 오버레이**. 레퍼런스 사진 위에 근육 구조를 반투명하게 겹쳐서 보여주면 피규어 원형 제작할 때 엄청나게 도움될 거예요.

**문하린(동인지):** **2인 포즈 검색**이요. 지금은 1인 마네킹만 있잖아요. 두 캐릭터가 상호작용하는 포즈(포옹, 싸움, 손잡기 등)를 검색할 수 있으면 BL/GL 동인 작가들한테 대박날 거예요.

**윤지아(UX):** UX 관점에서 **검색 히스토리 기반 추천 시스템**이요. "이 포즈를 자주 검색하시네요, 비슷한 레퍼런스 모음은 어떠세요?" 같은 개인화.

### 토론 4: "경쟁 서비스 대비 어떤가?"

**김도윤(게임):** Pinterest는 텍스트 검색만 되니까 비교 자체가 안 되고, PoseMyArt 같은 서비스는 3D 마네킹은 있는데 실사 레퍼런스 매칭이 없어요. **ArtRef는 3D 포즈 설정과 실사 검색을 처음으로 연결한 서비스**라고 봐요.

**이준호(3D):** SketchFab의 포즈 라이브러리는 3D 모델 자체를 제공하지만, "이 포즈의 실제 사진"은 안 줘요. ArtRef는 그 갭을 정확히 메워요.

**오태현(애니):** ShotDeck이랑 비교하면, ShotDeck은 영화 스틸컷 위주이고 월 $20인데, ArtRef는 인물 포즈에 특화되어 있고 3D 매칭이라는 독보적 기능이 있어요. 용도가 다르지만 ArtRef가 더 실용적이에요.

**한소율(초상화):** **Envato Elements**나 **Adobe Stock**은 키워드 검색이 전부인데, "이 각도에서 이 조명으로 찍은 인물" 같은 복합 검색이 안 돼요. ArtRef는 그게 되니까 차원이 다르죠.

**황민정(교사):** 교육용으로는 **경쟁자가 없어요**. 교육 모드(NSFW 필터) + 한글 태그 + 온보딩 튜토리얼 조합은 한국 미술 교육 시장에서 유일무이해요.

### 토론 5: "가격 정책에 대해"

**정민수(학생):** Free 50회/일은 과제 하나 하기에도 빡빡해요. **학생 플랜(월 3,000원에 무제한 검색)** 같은 거 만들어주세요.

**강현우(인디):** Free → Pro 사이가 너무 큰 갭이에요. **Lite 플랜** (월 5,000원, 검색 무제한 + 컬렉션 10개 + AI 추출 20회) 있으면 바로 결제해요.

**최유진(패션):** Team 플랜 가격이 어떻게 되는지 모르겠는데, **실시간 협업이 없으면 Team 플랜의 메리트가 약해요**. 단순히 공유 URL이면 Pro로 충분하거든요.

**오태현(애니):** 스튜디오 기준으로 **연간 결제 할인**이 있으면 예산 편성할 때 편해요. 월 만원짜리가 연간 10만원이면 바로 결재 올려요.

**임채원(강사):** **교육기관 라이선스** (학교 단위로 계정 50개 일괄 구매) 같은 B2B 모델도 고려해주세요. 교육청 예산으로 사려면 이런 패키지가 필요해요.

## PART 5: 대표 후기 (장문)

### 김도윤 (게임 컨셉 아티스트, 8년, Pro 플랜)

> 게임 업계에서 컨셉 아트를 그리면서 가장 시간을 많이 쓰는 게 레퍼런스 수집이에요. "왼쪽에서 측광을 받으면서 검을 들고 살짝 숙인 포즈"를 찾으려면 Pinterest에서 "sword pose side lighting"을 검색하고, 수십 페이지를 넘겨야 했거든요.
>
> ArtRef에서는 마네킹을 그 포즈로 잡고, 조명을 측광으로 세팅하면 3초 만에 딱 맞는 레퍼런스가 줄줄이 나와요. 이 경험이 너무 충격적이어서 팀원 5명한테 즉시 공유했습니다.
>
> 특히 카메라 앵글까지 맞출 수 있는 건 스토리보드 작업할 때도 유용해요. "로우앵글에서 올려다보는 영웅 포즈"를 정확하게 찾을 수 있거든요.
>
> 개선점이 있다면, 이미지 수가 아직 부족한 느낌이에요. 특정 포즈에서 매칭되는 이미지가 5장도 안 될 때가 있어요. Unsplash 수집을 계속 늘려주시면 좋겠습니다. 그리고 Photoshop 플러그인이 나오면 작업 흐름이 완전체가 될 것 같아요.

### 황민정 (중학교 미술교사, 5년, Free 플랜)

> 미술 수업에서 인체 드로잉을 가르칠 때 가장 곤란한 게 레퍼런스예요. Google 이미지 검색하면 수업에 부적절한 이미지가 섞여 나오고, 학부모 민원이 들어올까 봐 항상 조심스러웠거든요.
>
> ArtRef의 교육 모드를 발견하고 정말 감동했어요. "교육 모드" 버튼 하나로 완전히 안전한 이미지만 나와요. 학생들한테 "여기서 포즈 찾아보세요"라고 할 수 있게 된 게 처음이에요.
>
> 온보딩 튜토리얼도 잘 되어 있어서 중학생들도 5분이면 쓸 수 있어요. 한글 태그가 전부 한국어라서 "standing pose" 대신 "서있기"로 검색하면 되니까 영어를 못하는 학생도 문제없고요.
>
> 다만 무료 50회 검색은 수업 시간(45분)에 30명이 쓰기엔 턱없이 부족해요. 교육기관 라이선스로 반 전체가 쓸 수 있으면 교육청에 예산 신청할 의향이 있어요.

### 배준혁 (만화 어시스턴트, 1년, Free 플랜)

> 솔직히 컨셉은 좋은데 제 노트북(i5-8세대, 내장그래픽)에서는 사용이 거의 불가능해요. 마네킹 모드 들어가면 팬이 돌아가면서 프레임이 뚝뚝 끊기고, 관절 하나 잡는 데 3초씩 걸려요.
>
> 스케치 모드는 괜찮은데, 핵심인 3D 매칭을 못 쓰면 의미가 반감돼요. 2D 관절 인형 같은 심플 모드가 있으면 저사양에서도 핵심 기능을 쓸 수 있을 것 같아요.
>
> 만화 어시스턴트 월급으로 고사양 PC를 살 수도 없고, Pro 플랜을 결제하기도 부담스러워서... Lite 플랜이 나오면 당장 결제하겠습니다.
